# Decisions

## Static First Playable

Use a static Canvas prototype under `docs/` so the game can run on GitHub Pages immediately.

Godot 4 remains the intended engine for the full project, but the first playable combat loop is implemented as a web prototype to validate speed, dodge timing, chip strategy, hitstop, shake, and sound without export friction.

## Original IP Boundary

Use original robotics and cyber-infrastructure terminology. Avoid copied character, UI, story, and naming from existing works.
